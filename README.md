# parser_gui

목적 
영화 홈페이지 제작을 위한 네이버영화 홈페이지에서 영화 데이터 파싱하는 프로그램

### 개발환경
  * java8
  * 


### 기능목록 

   * 년도별 영화 데이터 xml/mysql 저장
   * 영화 포스터 저장
